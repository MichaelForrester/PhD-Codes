# Chapter 7 codes

- Lyapunov.m: Computes Lysapunov coefficient and rotation number for periodically forced next-generation neural-mass node.

- NextGen.m: Next-generation neural-mass node RHS.

- NextGenJ.m: Next-generation neural-mass node Jacobian.

- CBdelplot.m: Computes eigenvalue spectra for delayed and non-delayed next-generation neural mass networks.

- NextGenJ.m: Jacobian for the next-generation neural mass network.

- NG_delay.jl: Computes time series solutions for delayed and non-delayed next-generation neural mass networks.
